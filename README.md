# neeo-driver-nvidia-shield
Neeo remote driver to support the NVidia Shield via BlueTooth.

Keycode mapping
- Left          (80)
- Right         (79)
- Up            (82)
- Down          (81)
- Select        (40)
- Back          (27) / 29
- Home          (1,27)
- Volume up     (128) / 237
- Volume down   (129) / 238
- Mute          (127) / 239
- Menu          (101)
- Power on/off  (102)
- Play/pause    (232)
- Stop          (243)
- Fast forward  (242)
- Fast reverse  (241)
- Next song     (235)
- Previous song (234)

Modifier mapping:
- Left Ctrl     (1)
- Left Shift    (2)
- Left Alt      (4)
- Left Gui      (8)
- Right Ctrl    (16)
- Right Shift   (32)
- Right Alt     (64)
- Right GUI     (128)
