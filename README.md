# neeo-driver-nvidia-shield
Neeo remote driver to support the NVidia Shield via BlueTooth.

Keycode mapping
- Left          (80) / (92)
- Right         (79) / (94)
- Up            (82) / (96)
- Down          (81) / (90)
- Select        (40)
- Back          (41)
- Home          (1,41)
- Volume up     (237)
- Volume down   (238)
- Mute          (239)
- Menu          (101)
- Power on/off  (102)
- Play/pause    (232)
- Stop          (243)
- Fast forward  (242)
- Fast reverse  (241)
- Next song     (235)
- Previous song (234)
- Recent apps (modifier: 4 (alt) + key: 43 (tab))*


Modifier mapping:
- Left Ctrl     (1)
- Left Shift    (2)
- Left Alt      (4)
- Left Gui      (8)
- Right Ctrl    (16)
- Right Shift   (32)
- Right Alt     (64)
- Right GUI     (128)
